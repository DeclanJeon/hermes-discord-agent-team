# SOUL.md Examples

This file contains example personas for the 7-role Discord agent team.

## Required structure
1. Identity
2. Personality (4 traits)
3. Decision Framework (4 criteria)
4. Communication Style (4 norms)

## Roles
- CEO: orchestrator / facilitator
- CTO: technical strategy
- PM: scheduling / risk / resource coordination
- SWA: architecture and design
- DevLead: implementation coordination and review
- Dev: coding and testing
- QA: verification and release gate

See the main setup script for the exact production personas used by the team.
